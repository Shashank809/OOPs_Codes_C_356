# C++ Programs

A collection of C++ programs covering fundamental to intermediate concepts.

| Program | Description |
|---------|-------------|
| [Program1.cpp](Program1.cpp) | Demonstrates variable scoping, block scope, and the scope resolution operator (`::`) in C++. |
| [Program2.cpp](Program2.cpp) | Accepts two integers from the user and prints their sum. |
| [Program3.cpp](Program3.cpp) | Illustrates variable shadowing and scope resolution across nested blocks and `if` statements. |
| [Program4.cpp](Program4.cpp) | Uses an `inline` function to compute the cube of a number. |
| [Program5.cpp](Program5.cpp) | Demonstrates basic string output using `cout`. |
| [Program6.cpp](Program6.cpp) | Defines a function to add two integers and returns the result to `main`. |
| [Program7.cpp](Program7.cpp) | Compares two strings lexicographically using the `compare()` method. |
| [Program8.cpp](Program8.cpp) | Defines a `Rectangle` class with methods to display dimensions and calculate area. |
| [Program9.cpp](Program9.cpp) | Demonstrates single inheritance with a `Car` class inheriting `speed` from a `Vehicle` base class. |
| [Program10.cpp](Program10.cpp) | Creates a `Student` class with `id` and `name` attributes and a `display()` method. |
| [Program11.cpp](Program11.cpp) | Shows out-of-class member function definition using the scope resolution operator in an inheritance example. |
| [Program12.cpp](Program12.cpp) | Demonstrates accessing private base class data in a derived class through public getter/setter methods. |
| [Program13.cpp](Program13.cpp) | Defines a `Car` class with an out-of-class `show()` function to display brand and speed. |
| [Program14.cpp](Program14.cpp) | Demonstrates the `friend class` concept, allowing `Derived` to access private members of `Base`. |
| [Program15.cpp](Program15.cpp) | Reads student `id` and `name` from user input and displays them using an out-of-class method. |
| [Program16.cpp](Program16.cpp) | Demonstrates default arguments in a function by calculating rectangle area with optional parameters. |
| [Program17.cpp](Program17.cpp) | A multi-class banking system showcasing inheritance, friend functions, static members, constructors, and destructors. |
| [Program18.cpp](Program18.cpp) | Demonstrates parameterized and copy constructors using a `Base` class that stores student marks. |
| [Program19.cpp](Program19.cpp) | Illustrates single, multiple, and multilevel inheritance using three derived classes. |
| [Program20.cpp](Program20.cpp) | Simulates a bank account system demonstrating encapsulation, static members, and object interaction. |
| [Program21.cpp](Program21.cpp) | Uses abstract classes and pure virtual functions to calculate area of shapes (square and circle) via polymorphism. |
| [Program22.cpp](Program22.cpp) | Demonstrates runtime polymorphism using virtual functions and a base class pointer with two derived classes. |
| [Program23.cpp](Program23.cpp) | Illustrates the use of the `this` pointer to distinguish between class members and function parameters. |
| [Program24.cpp](Program24.cpp) | Shows how to access class members using both an object (`.`) and a pointer (`->`) operator. |
| [Program25.cpp](Program25.cpp) | Demonstrates base class pointer usage and explicit downcasting to call a derived class method. |
